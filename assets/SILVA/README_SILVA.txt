The SILVA database used here is generated from downloading the LSUParc and SSUParc rRNA databases from the SILVA project.

From here: https://www.arb-silva.de/no_cache/download/archive/current/Exports/, specifically:

These can be put together using 'cat *.fasta.gz > SILVA.fasta.gz' and later on indexed using bowtie2-index.

Please refer to bowtie2-index for further information.
